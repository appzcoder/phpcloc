# PHPCloc
Cloc & Code duplication checker written in PHP

## Requirements
    PHP >= 5.4

## Installation
```
$ composer global require appzcoder/phpcloc
```

## Usage
Cloc
```
$ phpcloc cloc .
```

Dublicate code checker
```
$ phpcloc duplicate . --ext=php
```
